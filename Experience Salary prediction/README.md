# Salary Prediction using Machine Learning

## Overview
This project builds a machine learning model to predict an employee’s **salary based on years of experience**. The goal is to understand the relationship between experience and salary growth using a regression model.

This project demonstrates the fundamentals of **data preprocessing, model training, and visualization** in a supervised learning problem.

---

## Problem Statement
Given the number of years of experience, predict the expected salary of an employee using a machine learning regression model.

---

## Dataset
The dataset contains two variables:

| Feature | Description |
|-------|-------------|
| YearsExperience | Number of years of professional experience |
| Salary | Annual salary of the employee |

---

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## Machine Learning Approach

The project uses **Linear Regression** to model the relationship between experience and salary.

Model Equation:

Salary = m × YearsExperience + b

Where:
- m = slope of regression line
- b = intercept

---

## Project Workflow

1. Import required libraries
2. Load and inspect dataset
3. Perform data visualization
4. Split dataset into training and testing sets
5. Train Linear Regression model
6. Generate predictions
7. Visualize regression results

---

## Results

The trained regression model successfully learns the relationship between **experience and salary growth**, allowing prediction of salary values for unseen experience levels.
