# Machine Learning Projects

This repository contains a collection of **Machine Learning and Deep Learning projects** implemented using Python.
These projects demonstrate practical applications of **data analysis, predictive modeling, and neural networks**.

---

## Projects

| Project                               | Description                                                                               |
| ------------------------------------- | ----------------------------------------------------------------------------------------- |
| **Experience Salary Prediction**      | Predicts salary based on years of experience using regression techniques.                 |
| **Placement Prediction**              | Predicts whether a student will get placed based on academic and skill-related features.  |
| **Titanic Survival Prediction**       | Predicts whether a passenger survived the Titanic disaster using a Scikit-Learn Pipeline. |
| **Time Series Forecasting with LSTM** | Forecasts airline passenger trends using a Long Short-Term Memory (LSTM) neural network.  |
