# Machine Learning Projects

This repository contains a collection of **Machine Learning and Deep Learning projects** implemented using Python.

It demonstrates practical applications of:
- Data Analysis  
- Predictive Modeling  
- Deep Learning  
- Real-world Problem Solving  

---

## 📌 Projects

| Project                               | Description                                                                 |
|--------------------------------------|-----------------------------------------------------------------------------|
| **Experience Salary Prediction**      | Predicts salary based on years of experience using regression techniques.   |
| **Placement Prediction**              | Predicts whether a student will get placed based on academic features.      |
| **Titanic Survival Prediction**       | Uses Scikit-Learn pipelines to predict survival on the Titanic dataset.     |
| **Time Series Forecasting (LSTM)**    | Forecasts airline passenger trends using LSTM neural networks.              |
| **YouTube Video Downloader & Analyzer** | Downloads YouTube videos and generates smart notes using AI/ML techniques. |

## 🚀 YouTube Video Downloader & Analyzer

### 🔹 Overview
This project allows users to:
- Download YouTube videos
- Extract useful content
- Generate **AI-based smart notes**

### 🔹 Features
- Video downloading functionality  
- Audio/Text extraction  
- Smart summarization  
- Clean and structured notes output  

### 🔹 Use Case
Ideal for:
- Students  
- Researchers  
- Content learners  
- Quick revision from long videos
